## How to use it (Replit)

1. Open **Shell** and apply:

   ```bash
   git checkout -b ui/desktop-card-header-reflow
   printf "%s\n" 'PASTE THE PATCH CONTENTS HERE' > desktop-card-header-reflow.patch
   git apply --reject --whitespace=fix desktop-card-header-reflow.patch
   git add -A
   git commit -m "Desktop Patient Card: header reflow, 2-line name, EWS+ATS moved to meta row, location pill"
   ```

2. **Pass a real location label** when you render the card:

   ```tsx
   <PatientCardExpandable
     patientId={p.id}
     name={p.displayName}
     status={lane.label}
     timer={p.waitingFor}
     complaint={p.chiefComplaint}
     ats={p.ats}
     ageSex={p.age ? `${p.age}${p.sex ? ` ${p.sex}` : ""}` : p.sex}
     statusFlags={{
       isolation: p.isolation,
       allergy: !!(p.allergies?.length),
       oxygen: p.onOxygen === true,
       sepsis: p.sepsisAlert === true,
       tasksDue: p.tasksDueCount,
       resultsPending: p.resultsPendingCount,
     }}
     // NEW — feed the actual physical location label here:
     locationLabel={
       p.roomName
         ? p.roomName                      // e.g., "Cubicle 7" or "Room 3"
         : p.treatmentArea
           ? `${p.treatmentArea}`          // e.g., "RESUS 2", "OBS 5", "ISO 1"
           : undefined
     }
   />
   ```

3. **Result**

   * Name now uses a **two-line clamp** on desktop (no weird truncation).
   * **EWS (live)**, **ATS**, **Timer**, **Last obs** moved to a calm **meta row**.
   * A clear **Location** pill (with bed icon) shows **where the patient physically is**: “RESUS 2”, “OBS 5”, “Cubicle 7”, etc.
   * **StatusStrip** wraps gracefully on the right, so it won’t crush the name.
