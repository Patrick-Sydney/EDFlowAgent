### What you’ll see

* **Tablet & Desktop** now use the **same compact header** as mobile.
* The **app logo opens one dropdown** that contains **Role** (compact) and **Scenarios**.
* The separate Role/Scenarios controls are removed on large screens, preventing crowding and ensuring consistent behavior.

If you want the **title/tagline** visible again on desktop but still keep the combined menu, I can add a tiny, muted title that only appears at `xl+` without increasing header height.
